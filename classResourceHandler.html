Detailed Description

template<typename Key, resourceDerived ResourceType>
class ResourceHandler< Key, ResourceType >

Data structure that handles the references to a specific type of resource and its memory.

Template Parameters
KeyType used to access the resources.
ResourceTypeType of the resource to handle, must be a derived class from Resource.
